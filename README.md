# Mobile Device Usage Explorer

This Shiny app allows users to explore patterns in mobile phone usage behavior
based on demographics and daily activity.

### Features
- Sidebar filters for gender, education, screen time, and sleep hours
- Dynamic data subset and visualizations
- Download filtered data as CSV
- Summary tables and correlation heatmap

### Dataset
[Mobile Device Usage and User Behavior Dataset (Kaggle)](https://www.kaggle.com/datasets/valakhorasani/mobile-device-usage-and-user-behavior-dataset)
